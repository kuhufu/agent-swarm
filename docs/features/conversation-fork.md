# 会话分支

会话分支用于从已有对话创建一个新的独立会话，便于在不破坏原始历史的情况下继续调试 prompt、模型或 Swarm 配置。

## 接口

```http
POST /api/conversations/:id/fork
```

请求体：

```json
{
  "title": "可选标题",
  "swarmId": "可选目标 Swarm",
  "messageId": "可选消息 ID"
}
```

- 未传 `messageId` 时，复制源会话的完整消息历史。
- 传入 `messageId` 时，只复制从开头到该消息为止的历史。
- 新会话会继承源会话的 `enabledTools`、`thinkingLevel` 和直接对话模型偏好。
- 如果源会话存在上下文清空边界，且该边界仍位于被复制的历史范围内，新分支会保留该边界。

## 前端入口

- 聊天页顶部“分支”按钮：从当前会话完整历史创建分支。
- 消息 hover 后的分支按钮：从该消息处创建分支，并跳转到新会话。

分支创建后，原会话不会被修改。
